## Evaluation is an artifact problem

A local adapter should not be delivered as a naked file. It should include the training configuration, base model reference, dataset version, task tests, regression results, limitation notes, and activation instructions.

## Minimum evidence pack

- Dataset manifest
- Training run record
- Task evaluation suite
- Regression report
- Known limitations
- Deployment notes
